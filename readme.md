# peer-to-peer-chat

Simple peer-to-peer chat application

## Usage

#### Install

```sh
$ npm install
```

#### Architecture

peer1 <---> peer2 <---> peer3 

#### Connect peer1 peer2 and peer3 

```sh
$ node peer.js peer1 peer2

$ node peer.js peer2 peer3

$ node peer.js peer3
```
